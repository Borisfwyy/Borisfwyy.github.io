CMD

CMD命令

开启使用

WIN+R打开运行输入cmd即可进入命令行
也可以通过cmd /c 命令 和 cmd /k 命令的方式来直接运行命令
/c表示执行完命令后关闭cmd窗口；/k表示执行完命令后保留cmd窗口

切换目录

  • 从C盘切换到其他盘符：直接输入相应盘符回车即可
    形如：“F:”
  • cd命令：cd为切换目录命令
    命令 效果
    cd 显示当前目录
    cd .. 返回上一层目录
    cd JAVA\java 进入当前目录下的JAVA\java文件夹

其他命令

  1. cls：清除当前屏幕显示
  2. 复制内容：右键弹出快捷菜单，选择“标记(K)”，然后选中所需复制的内容，然后右键即可
  3. 粘贴内容：右键弹出快捷菜单，选择“粘贴(P)”
  4. 在文件夹空白处按住Shift，然后右键弹出快捷菜单，可以看到“在此处打开命令行窗口”
  5. 使用上下方向键，翻看使用过的命令
  6. tab补齐功能
  7. 文件及目录名中不能包含下列任何字符：\ / : * ? " < > |
