HTML使用手册

一个简化的 HTML 文档示例：

<!DOCTYPE html>
<html>
<head>
<title>Title of the document</title>
<base href="..." />
<base target="_blank" />
<link rel="stylesheet" type="text/css" href="mystyle.css" />
</head>
<body>
The content of the document......
</body>
</html>

This is a link to the page ...

HTML标签

  • HTML 标签是由尖括号包围的关键词，比如 <html>
  • HTML 标签通常是成对出现的，比如 <b> 和 </b>
  • 标签对中的第一个标签是开始标签，第二个标签是结束标签
  • 开始和结束标签也被称为开放标签和闭合标签

HTML 标题

HTML 标题（Heading）是通过 <h1> - <h6> 等标签进行定义的。

数字1-6表示标题的大小，数字越小，字符越大

成对出现

HTML段落

HTML 段落是通过 <p> 标签进行定义的。成对出现

换行使用<br>

HTML链接

HTML 链接是通过 <a> 标签进行定义的。

在 href 属性中指定链接的地址。

成对出现

  • 示例：This is a link
  • <a href="http://www.bilibili.com.cn">This is a link</a>

HTML 图像

HTML 图像是通过 <img> 标签进行定义的。

  • 图像的名称和尺寸是以属性的形式提供的。

  • 可以使用网络图片地址

  • 形如：<img src="eu4-2.jpg" width="100" height="150" />

HTML表格

  • 表格由<table>标签来定义。每个表格均有若干行（由 <tr> 标签定义），每行被分割为若干单元格（由 <td>标签定义）。字母 td 指表格数据，即数据单元格的内容。

  • 数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。

  • 如果不定义边框属性，表格将不显示边框。使用边框属性<table border="1">来显示一个带有边框的表格

  • 表格的表头使用<th>标签进行定义，可以定义多个表头

  • 显示空单元格时，为避免显示错误，需要在空单元格中加入一个空格

  • Heading Another Heading
    row 1, cell 1 row 1, cell 2
    row 2, cell 2 

<table border="1">
<tr>
<th>Heading</th>
<th>Another Heading</th>
</tr>
<tr>
<td>row 1, cell 1</td>
<td>row 1, cell 2</td>
</tr>
<tr>
<td> </td>

</tr>
</table>

HTML样式

  • 通过把style=""嵌入到其他标签中

  • 可以定义字体颜色和尺寸，具体类型可以在网上搜索

HTML无序列表

  • 无序列表是一个项目的列表，此列项目使用粗体圆点（典型的小黑圆圈）进行标记。

  • 无序列表始于 <ul> 标签。每个列表项始于 <li>
    <ul>
<li>Coffee</li>
<li>Milk</li>
</ul>

HTML有序列表

  • 有序列表也是一列项目，列表项目使用数字进行标记

  • 有序列表始于 <ol> 标签。每个列表项始于<li>标签

    <ol>
<li>Coffee</li>
<li>Milk</li>
</ol>

HTML块

  • 大多数 HTML 元素被定义为块级元素或内联元素
  • 例子：<h1>, <p>, <ul>, <table>

  • HTML <div> 元素是块级元素，它是可用于组合其他 HTML 元素的容器。
  • 如果与 CSS 一同使用，<div>元素可用于对大的内容块设置样式属性。
  • <div> 元素的另一个常见的用途是文档布局。

HTML背景

<body> 拥有两个配置背景的标签。背景可以是颜色或者图像

HTML背景颜色

背景颜色属性将背景设置为某种颜色。属性值可以是十六进制数、RGB 值或颜色名

  • 黑色示例：
    <body bgcolor="#000000">
    <body bgcolor="rgb(0,0,0)">
    <body bgcolor="black">

HTML背景图像

背景属性将背景设置为图像。属性值为图像的URL。如果图像尺寸小于浏览器窗口，那么图像将在整个浏览器窗口进行复制。

  • 相对地址：<body background="clouds.gif">
  • 绝对地址：<body background="http://www.w3school.com.cn/clouds.gif">

HTML头部元素

HTML <head> 元素

  • <head> 元素是所有头部元素的容器。<head>内的元素可包含脚本，指示浏览器在何处可以找到样式表，提供元信息，等等。
  • 以下标签都可以添加到 head 部分：<title>、<base>、<link>、<meta>、<script> 以及 <style>

HTML <title> 元素

<title> 标签定义文档的标题。

  • title 元素在所有 HTML/XHTML 文档中都是必需的。
  • title 元素能够：
    定义浏览器工具栏中的标题
    提供页面被添加到收藏夹时显示的标题
    显示在搜索引擎结果中的页面标题

HTML<base> 元素

<base> 标签为页面上的所有链接规定默认地址或默认目标

HTML <link> 元素

<link> 标签定义文档与外部资源之间的关系,最常用于连接样式表：

HTML <style> 元素

<style> 标签用于为 HTML 文档定义样式信息。

HTML<script> 元素

<script> 标签用于定义客户端脚本，比如 JavaScript。
