Gitbook

Gitbook制作

前期准备

  • 命令操作基于CMD，若不能正常使用请切换管理员模式

安装Node.js

  • GitBook 是一个基于 Node.js 的命令行工具，下载安装 Node.js
  • 安装完成之后，你可以使用命令node -v来检验是否安装成功
  • 若安装成功，系统会显示Node.js的版本，如v7.7.1

安装 GitBook

  • 输入命令npm install gitbook-cli -g来安装 GitBook
  • 安装完成之后，你可以使用命令gitbook -V来检验是否安装成功
    gitbook -V
CLI version: 2.3.2
GitBook version: 3.2.3

制作文档

创立文档

输入命令gitbook init，得到以下代码：

gitbook init
warn: no summary file in this book
info: create README.md
info: create SUMMARY.md
info: initialization is finished

可以看到他会创建 README.md和 SUMMARY.md这两个文件

  • README.md就是说明文档
  • SUMMARY.md就是书的章节目录，其默认内容如下所示：
    # Summary
    * [Introduction](README.md)

  • 输入gitbook serve 命令，然后在浏览器地址栏中输入系统给与的地址便可预览书籍
  • 输入gitbook build 命令，可以生成文件，但不能直接使用
